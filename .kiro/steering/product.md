# Product Definition: Akyobox

## Vision
A centralized portal for "Akyo" themed web games, starting with the "Whack-a-Akyo" (Whack-a-Devil-Yagi-Akyo) Unity WebGL game. The platform is built on Next.js 16 to ensure high performance, scalability, and ease of future expansion.

## Core Features
1.  **Game Portal**: A landing page hosting the embedded Unity game.
2.  **Unity Integration**: Seamless embedding of high-quality Unity 6 WebGL builds.
3.  **Optimized Delivery**: Support for Brotli-compressed assets for fast loading times.
4.  **Future Expansion**: Ready to host multiple games, leaderboards, and user accounts.

## Target Audience
- Fans of the Akyo character.
- Casual browser game players.
