# よく使うコマンド (Windows / PowerShell または CMD)
- 開発サーバー: `npm run dev`  (Next.js 16、Node >=20.9)
- 本番ビルド: `npm run build`
- 本番起動: `npm start`
- Lint: `npm run lint`
- Dead code チェック: `npm run deadcode` (knip)
- 依存インストール: `npm install`
- Unity ビルド成果物配置: Unity 側でビルド後、生成物を `public/games/whack-a-devilyagiakyo/` にコピー（.br 生成しない）。
